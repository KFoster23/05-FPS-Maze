# 05-FPS-Maze
Fifth Project for MSCH-C220, 10 April 2021

A Maze that with multiple routes, a hidden key, and an enemy.
# Implementation
Built using Godot 3.2.3 Includes WASD mappings (up, left, down, right) and escape to quit.
